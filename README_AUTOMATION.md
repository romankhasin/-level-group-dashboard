# Автоматическое обновление дашборда

Workflow `.github/workflows/update-data.yml` запускается ежедневно в 06:00 по Москве и вручную через вкладку Actions.

Он обновляет:

- Яндекс Метрику по счётчикам `53197618` и `100470605`: с 1 мая 2026 года до вчерашнего дня, далее — только недостающие дни;
- показы, клики и стоимость Yandex/MTS из вкладки `Данные_метрика` по ключу `technical_mark`;
- `journal.html` из вкладки `Дневник оптимизации`, без фильтра по площадкам;
- Target Ads после появления секрета `TARGETADS_TOKEN`.

Обязательный Repository secret: `YANDEX_METRIKA_TOKEN`.

Необязательный Repository secret: `TARGETADS_TOKEN`. Пока его нет, остальные источники продолжают обновляться.
